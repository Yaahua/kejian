# 新编中国历史文选 — 课件导航

> 北京大学历史学系 · 何晋老师 · 本科生核心课程

## 简介

本仓库为北京大学何晋老师「中国历史文选」课程的全部课件数字化存档，基于原课件站（http://ftp.hist.pku.edu.cn/person/hejin/zglswx/）重新整理。

内容涵盖**考古文献、经部、史部、子部、集部**五大板块，含课文选读、专题索引、补充资料、教学参考等。

## 在线浏览

访问 [https://yaahua.github.io/kejian/](https://yaahua.github.io/kejian/)

## 编码说明

课件原始文件使用 **Big5/GBK** 编码（繁体中文）。若浏览器显示乱码，请右键 → 编码 → 选择「繁体中文(Big5)」或「中文(简体)」。

## 目录结构

| 文件夹 | 内容 |
|--------|------|
| `00_kaogu/` | 考古文献（甲骨文、金文、简牍、敦煌文书等）|
| `01_jingbu/` | 经部（周易、尚书、诗经、周礼、左传等）|
| `02_shibu/` | 史部（汉书、三国志、明史、资治通鉴等）|
| `03_zibu/` | 子部（庄子、吕氏春秋、日知录等）|
| `04_jibu/` | 集部（韩昌黎文集等）|
| `fulu/` | 附录（简化字总表等）|
| `download/` | 古籍全文下载区 |
| `images/` | 图片素材 |
